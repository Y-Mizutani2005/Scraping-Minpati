# Scraping-Minpati
遊戯機情報サイトから約一万件の情報を抽出するときに書いたプログラムです。
<br>
FullData.csvが抽出した結果。
<br>
目的の詳細：某遊戯機メーカーで勤務している父からの要望「みんぱちというサイトで使われている台データの参照元を知りたい」<br>
やったこと：当該サイトからスクレイピングを行い、ホールごとにリンク元のURLを抽出し参照元を特定
